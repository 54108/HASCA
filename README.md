## 说明
### 解压后数据应当按照如下格式组织

```
data_gen
├── data
│   ├── test
│   ├── train
│   └── val
└── raw_data
    ├── example-submission
    ├── test
    ├── train
    │   ├── Bag
    │   ├── Hand
    │   ├── Hips
    │   └── Torso
    └── val
        ├── Bag
        ├── Hand
        ├── Hips
        └── Torso
```
